Particle Button Module

A "Plug & Play" Explosion Component

Overview

The ParticleButtonModule is a lightweight, dependency-free JavaScript module that creates interactive buttons with SVG particle explosions. It is designed using the Revealing Module Pattern to ensure zero conflicts with other scripts.

1. Installation

Drop the particle-button.js and particle-button.css files into your project folder and link them in your HTML.

<link rel="stylesheet" href="particle-button.css">
<script src="particle-button.js"></script>

2. Basic Usage (Standalone)

To create a button, provide a target ID and the button text.

ParticleButtonModule.create('container-id', 'Click Me!');

3. Advanced Usage (Custom Variations)

You can customize the colors, particle density, and behavior via the options object.

ParticleButtonModule.create('target-id', 'Delete Item', {
    colors: ['#ff0000', '#000000'], // Red & Black
    count: 60,                     // More particles
    onClick: () => {
        console.log("Action performed!");
    }
});

4. Integrated Usage (With Other Modules)

This module is perfect for triggering side effects like notifications or sound effects.

ParticleButtonModule.create('reward-btn', 'Claim Prize', {
    onClick: () => {
        // Calling your Notification Module
        NotificationModule.notify('Success', 'Points Added!', { 
            type: 'success', 
            sound: 'ping.mp3' 
        });
    }
});

Options Reference

Property Type Default Description
colors Array Mixed Pink/Blue Array of HEX or RGB color strings.
count Number 30 The number of particles to spawn per click.
onClick Function null The logic to execute after the explosion starts.
